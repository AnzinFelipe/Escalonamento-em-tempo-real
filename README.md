# Escalonamento em Tempo Real

## Como compilar e executar

para compilar escreva:

- _make clean_
- _make rate_ ou _make edf_

para executar:

- _./rate [namefile]_ ou _./edf [namefile]_

para testar:

- *make test_rate* ou *make test_edf*
